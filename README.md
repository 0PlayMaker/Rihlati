# 🌸 رحلتي

### تطبيق تتبّع يومي شامل، خاص تمامًا، بلا إنترنت وبلا سيرفر

**رحلتي** تطبيق ويب تقدمي (PWA) صُمم بمحبة لتتبّع كل تفاصيل يومك — من عاداتك وعباداتك، إلى صحتك وطعامك واقتصادك ويومياتك — كله في مكان واحد، هادئ وبسيط وجميل.

كل بايت من بياناتك يبقى على جهازك أنت. لا حساب، لا تسجيل دخول، لا سيرفر خارجي، لا أي جهة ثالثة ترى ولو حرفًا واحدًا مما تكتبينه.

---

## ✨ لماذا رحلتي؟

أغلب تطبيقات التتبّع في المتاجر تطلب حسابًا، وتُرسل بياناتك لسيرفرات لا تعرفين مكانها، وتتوقف عن العمل إذا أوقفت الشركة الخدمة. **رحلتي** مختلف تمامًا:

- 🔒 **خصوصية كاملة** — كل البيانات محفوظة محليًا على جهازك (IndexedDB)، ولا تغادره أبدًا
- 📴 **يعمل بدون إنترنت بالكامل** — بعد أول تحميل، لا تحتاجين اتصالًا إطلاقًا
- 🆓 **مجاني ومفتوح المصدر** — استضيفيه بنفسك مجانًا على GitHub Pages
- 📱 **تجربة تطبيق حقيقية** — أضيفيه لشاشة هاتفك الرئيسية ويعمل كأي تطبيق آخر
- 🌙 **تصميم عربي أصيل** — من اليمين لليسار بالكامل، بخط Tajawal، وألوان هادئة

## 🧩 الأقسام والميزات

### 🌱 العادات والمهام
- عادات جيدة تبنينها، وعادات سيئة تُقلعين عنها — بنفس نظام التتبّع لكن بمعنى معكوس
- تسلسل الأيام (streak)، وعدد الأيام الناجحة والفائتة لكل عادة
- مهام يومية ثابتة (بتذكير اختياري) ومهام لمرة واحدة، منفصلة وواضحة
- تعديل وحذف سهل لأي عنصر عبر قائمة ⋮ أنيقة

### 🕌 العبادة
- الصلوات الخمس المفروضة، مع تسلسل يومي لكل صلاة
- السنن، الأذكار بعد الصلاة، أذكار الصباح والمساء
- أذكار مخصصة تضيفينها بنفسك، بعدّاد تكرار يومي

### 🌙 الدورة الشهرية
- تسجيل بداية ونهاية الدورة، مع تقويم يوضح الفترة بصريًا
- توقّع الدورة القادمة بناءً على متوسط دوراتك الفعلي (وليس رقمًا عامًا مفترضًا)
- إيقاف تلقائي لتتبّع الصلاة أثناء الدورة

### ⚖️ الصحة
- وزن يومي برسم بياني تفاعلي (اضغطي على أي نقطة لرؤية تاريخها ووزنها بالضبط)، مع اختيار المدى الزمني (شهر، شهرين، سنة، الكل)
- مؤشر كتلة الجسم كمرجع عام، وليس حكمًا نهائيًا
- قياسات جسم مخصصة (خصر، ذراع، أي شيء تريدين قياسه)
- 💪 **قسم تمارين كامل**: تمارين بوصف نصي وصورة ورابط يوتيوب، عدّاد مجموعات، تسلسل أيام، و**مؤقّت حقيقي يعمل** (تنبيه صوتي واهتزاز عند الانتهاء)
- مزاج يومي بإيموجي، مع سجل كامل وشريط آخر ٧ أيام في الصفحة الرئيسية

### 🍽️ الطعام
- تسجيل الوجبات (فطور، غداء، عشاء، سناك، تحلية) بصورة وسعرات اختيارية
- متتبّع ماء يومي بهدف قابل للتعديل
- 📖 **دفتر وصفات**: احفظي وصفاتك بفيديو يوتيوب، صور، مكونات، وطريقة التحضير

### 🎯 الأهداف
- ثلاثة أنواع: بسيط (✅/❌)، رقمي (تقدّم فعلي، صاعد أو هابط — مثل "نزول من ٧٠ إلى ٦٠ كغم")، أو نسبة مئوية بشريط تمرير
- يفهم تلقائيًا اتجاه هدفك دون الحاجة لتحديده يدويًا

### 📔 يومياتي
- يومية واحدة لكل يوم، بصورة اختيارية، وتسلسل أيام الكتابة
- كل اليوميات مرتبة تلقائيًا حسب الشهر في قوائم قابلة للطي

### 💰 الاقتصاد
- رصيد محسوب تلقائيًا من سجل معاملاتك (وليس رقمًا منفصلًا قد يختل)
- معاملات دخل ومصروف، وعملة تختارينها بنفسك (دينار، ريال، أي شيء)
- 🛒 قوائم تسوق متعددة بخانات اختيار، قابلة للحذف
- 🍎 مأكولات تشترينها (بخصم تلقائي اختياري من الرصيد) + قائمة أمنيات مأكولات
- 🛍️ أغراض تشترينها، بنفس الطريقة تمامًا + قائمة أمنيات أغراض

### 📊 نظرة على عامك
- ملخص سنوي شامل لكل قسم: عادات، مهام، عبادة، مزاج، دورة، طعام، صحة، أهداف، يوميات، اقتصاد
- تنقّل بين السنوات بسهولة

### ⚙️ إضافات عامة
- 🎨 تخصيص المظهر بالكامل: لون تمييز تختارينه بنفسك، وأربعة أوضاع (فاتح، داكن، أسود عميق موفّر للبطارية، وزجاجي شفاف)
- نسخ احتياطي واستعادة كاملة (ملف .zip واحد يشمل كل شيء، حتى الصور)
- شريط تنقّل سفلي قابل للتخصيص بالكامل (اختاري أي أقسام تظهر)
- رمز حماية (PIN) محلي، بدون أي اتصال بالإنترنت
- عبارات ترحيب يمكنك إضافتها أو تعديلها بنفسك

## 🛠️ التقنيات المستخدمة

مبني بالكامل بتقنيات ويب أساسية، بدون أي إطار عمل (framework) ثقيل:

| التقنية | الاستخدام |
|---|---|
| HTML / CSS / JavaScript (Vanilla) | كل واجهة التطبيق ومنطقه |
| [Dexie.js](https://dexie.org) | طبقة فوق IndexedDB لتخزين البيانات محليًا |
| [JSZip](https://stuk.github.io/jszip/) | تجميع النسخة الاحتياطية في ملف .zip واحد |
| Service Worker | يجعل التطبيق يعمل بالكامل بدون إنترنت |
| Web Manifest | يسمح بتثبيت التطبيق على الشاشة الرئيسية |

لا Node.js، لا React، لا Build Step — افتحي `index.html` وابدئي.

## 🚀 التثبيت والنشر

### تجربة سريعة محليًا
افتحي `index.html` مباشرة في المتصفح، أو شغّلي سيرفرًا محليًا بسيطًا:
```bash
python3 -m http.server 8000
```

### النشر على GitHub Pages (الأسهل، ومجاني)
1. ارفعي كل ملفات مجلد `rahlati/` إلى مستودع GitHub جديد
2. من إعدادات المستودع (Settings) ← Pages ← اختاري النشر من فرع `main` والمجلد الرئيسي
3. بعد دقيقة، سيكون تطبيقك متاحًا على رابط مثل: `https://username.github.io/rahlati/`
4. من متصفح الجوال، اختاري "إضافة إلى الشاشة الرئيسية" ليعمل كتطبيق حقيقي

---

## 🔐 الخصوصية أولًا، دائمًا

لا يوجد سيرفر خلفي (backend) في هذا المشروع إطلاقًا. كل بيانات المستخدم — العادات، الصور، الوزن، الدورة، كل شيء — تُخزَّن فقط داخل متصفح جهاز المستخدم عبر IndexedDB. لا شيء يُرسل لأي مكان، أبدًا.

النسخة الاحتياطية الوحيدة الموجودة هي ملف تُنزّلينه أنتِ بنفسك، وتحتفظين به أينما شئتِ.

- إصلاح: صورة الطعام كانت تفتح الكاميرا مباشرة بدون خيار المعرض، بعكس كل صور التطبيق الأخرى

- بدء/إنهاء الدورة أصبح يسأل عبر مربع اختيار إذا كانت تريد إيقاف/استئناف تتبع الصلوات، بدلاً من فعل ذلك تلقائياً دون اختيار
